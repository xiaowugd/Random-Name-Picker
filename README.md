<h3 align="center">简洁美观的网页随机点名工具</h3>

<p align="center">
  一款界面美观、交互流畅的在线随机点名工具，支持分组管理、多语言切换与自动深色模式。
</p>

---

## 🧭 中文介绍

### 🚀 特点亮点

- 🌐 **中英文切换**：支持中英文界面，记忆用户语言偏好，跨设备一致体验。
- 🌗 **自动深色模式**：根据系统主题自动切换，白天清爽，夜间护眼。
- 📑 **分组管理系统**：支持多分组添加、重命名、删除，名单分类更清晰。
- ✨ **平滑动画过渡**：点名过程采用渐变与缩放动画，自然不突兀。
- 🎛️ **双模式点名**：可自动点名或手动控制，灵活应对不同场景。
- 💾 **本地存储支持**：所有数据本地存储，隐私友好，无需联网或登录。
- 📱💻 **响应式设计**：适配手机、平板与电脑，UI 自适应布局流畅自然。
