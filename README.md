## 🧭 中文介绍

### 🚀 特点

- 🌗 **自动深色模式**：根据系统自动切换主题，提升视觉舒适度。
- 📑 **分组管理**：快速创建、修改和删除分组，轻松管理各类名单。
- ✨ **动态动画效果**：流畅自然的随机点名动画，增强用户体验。
- 🎛️ **双模式点名**：支持自动随机与手动停止两种点名模式。
- 💾 **本地数据存储**：无需安装，数据存储在本地，隐私安全。
- 💻📱 **自适应布局**：完美适配电脑、平板与手机，界面灵活美观。
