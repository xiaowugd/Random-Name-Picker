## 🧭 English Introduction

### 🚀 Features

- 🌗 **Automatic Dark Mode**: Automatically switches themes based on the system setting.
- 📑 **Group Management**: Create, rename, and delete groups to organize names easily.
- ✨ **Smooth Animations**: Visually appealing random draw with fluid motion.
- 🎛️ **Dual Modes**: Supports both auto-pick and manual stop.
- 💾 **Local Data Storage**: Runs entirely in your browser, private and secure.
- 💻📱 **Responsive Design**: Fully adapts to desktop, tablet, and mobile.
